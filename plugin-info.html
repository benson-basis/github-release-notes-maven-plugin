Plugin Documentation

Goals available for this plugin:

Goal Description
github-release-notes:help Display help information on github-release-notes-maven-plugin.
Call mvn github-release-notes:help -Ddetail=true -Dgoal=<goal-name> to display parameter details.
github-release-notes:write-release-notes Write release notes file to github's release/tag structure.

System Requirements

The following specifies the minimum requirements to run this Maven plugin:

Maven 3.0.4
JDK 1.6
Memory No minimum requirement.
Disk Space No minimum requirement.

Usage

You should specify the version in your project's plugin configuration:

<project>
  ...
  <build>
    <!-- To define the plugin version in your parent POM -->
    <pluginManagement>
      <plugins>
        <plugin>
          <groupId>com.basistech</groupId>
          <artifactId>github-release-notes-maven-plugin</artifactId>
          <version>0.0.1-SNAPSHOT</version>
        </plugin>
        ...
      </plugins>
    </pluginManagement>
    <!-- To use the plugin goals in your POM or parent POM -->
    <plugins>
      <plugin>
        <groupId>com.basistech</groupId>
        <artifactId>github-release-notes-maven-plugin</artifactId>
        <version>0.0.1-SNAPSHOT</version>
      </plugin>
      ...
    </plugins>
  </build>
  ...
</project>

For more information, see "Guide to Configuring Plug-ins"
